# Tutorials

With many examples on common PHP tools, discover this plugin features :

## How to use the `binary-directory` option

With [Easy-Coding-Standard](ecs.md),
we will see how to specify the binary command folder (`XDG_BIN_HOME` environment variable)
of your favorite CLI tools.

> [!TIP]
>
> If you prefer install your tools with PHIVE rather than Composer, have a look on how to use the `dependency-manager` option.

## How to use the `config-directory` and `config-file` options

With [PHPUnit](phpunit.md),
we will see how to specify the configuration file lookup directory (`XDG_CONFIG_HOME` environment variable)
of your favorite CLI tools.

## How to use the `colors` option

- With [Mago](mago.md),
we will see how to add automagically `--colors=<when>` flag syntax (**when** = `auto`|`always`|`never`)
to your `action` definition line.

- With [PHPLint](phplint.md),
we will see how to add automagically `--ansi` flag syntax (common to all Symfony Console tools)
to your `action` definition line.

## How to use the `dependency-manager` option

With a [PHP_CodeSniffer](phpcs.md) PHAR distribution, we will see how to use your CLI tools installed by PHIVE.
